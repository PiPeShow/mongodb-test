MongoDB Test

1. What is MongoDB?

A) A relational database
B) A NoSQL database
C) A type of SQL
D) None of the above

MongoDB is a NoSQL database that uses a document-oriented data model. It is designed to handle large volumes of unstructured data efficiently.

2. Which of the following is a feature of MongoDB?

A) Strongly typed schemas
B) Flexible schema
C) Fixed data types
D) None of the above

MongoDB allows for a flexible schema, meaning fields can vary from document to document and data types can change.

3. What is the format of data storage in MongoDB?

A) JSON-like format
B) BSON format
C) XML format
D) CSV format

MongoDB stores data in BSON (Binary JSON) format, making it easy to map to objects in programming languages.

4. Which of the following is true regarding MongoDB collections?

A) Collections can only store data of the same type
B) Collections can store documents of different shapes
C) Collections are like tables in SQL
D) None of the above

In MongoDB, a collection can contain documents of varying structures, allowing flexibility in data storage.

5. What command is used to insert a document in MongoDB?

A) addDocument()
B) insertOne()
C) createDocument()
D) None of the above

The method insertOne() is used to insert a single document into a collection in MongoDB.
